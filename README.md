# Design and Implementation of a Regulated 12V–5V Power Supply with Arduino-Based Temperature Monitoring and Automatic Fan Control

## Description
This project focuses on designing a regulated 12V to 5V power supply for automotive applications. It also includes temperature monitoring using Arduino and automatic fan control to prevent overheating.

## Objectives
- Convert 12V car battery to stable 5V
- Monitor temperature of engine system
- Automatically control cooling fan
- Ensure safe operation

## Features
- Voltage regulation (12V → 5V)
- Temperature sensing
- Automatic fan ON/OFF control
- Reliable automotive application

## Technologies Used
- Arduino
- Voltage Regulator (7805 / Buck Converter)
- Temperature Sensor (LM35)
- Embedded C

## Hardware Components
- Arduino
- Voltage Regulator
- Temperature Sensor
- Fan
- Power Supply Circuit

## Working Principle
The system converts 12V input into a stable 5V output using a voltage regulator. The temperature sensor continuously monitors the engine temperature. When the temperature exceeds a threshold, the Arduino activates the fan automatically.

## Results
The system successfully maintains stable voltage and controls temperature using automatic fan operation.

## Future Scope
- Add PWM-based fan speed control
- Improve efficiency using buck converter
- Add display for temperature monitoring
